# Bill of Material

Parts list for the Hawkridge package. The visible PDF version is attached in the Printables Documentation section: https://www.printables.com/model/1776629-esp32-c3-co2-monitor-enclosure

| Item | Qty | Category | Notes |
|---|---:|---|---|
| ESP32-C3 SuperMini board | 1 | Electronics | Upstream README recommends ESP32-C3 SuperMini for the enclosure. |
| Sensirion SCD41 CO2 sensor module | 1 | Sensors | README lists SCD41 for CO2, temperature, and humidity measurement. |
| BMP180 pressure/temperature sensor module | 1 | Sensors | README lists BMP180 for barometric pressure and temperature. |
| 1.28 inch round TFT display, 240x240, GC9A01 | 1 | Display | README specifies a 1.28 inch round 240x240 TFT display with GC9A01 driver. |
| Passive buzzer module | 1 | Alerts | Pinout/config includes a buzzer on GPIO20 for CO2 alerts. |
